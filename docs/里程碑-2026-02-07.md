## 如何手工验证（本地）
1. 启动接收器（默认 TCP 8441、UDP 4565）：
   - 运行 `artifacts/TestReceiverPublish/TestReceiver.exe`
2. 启动 UI：
   - 运行 `artifacts/SimulatorAppPublish/SimulatorApp.exe`
3. UI 内建议配置：
   - Host 使用 `127.0.0.1`（避免 localhost IPv6/IPv4 差异）
   - Port 使用默认值
4. 依次点击：并发注册 → 开始心跳 / 日志发送“添加任务” / 白名单上传“添加任务”
