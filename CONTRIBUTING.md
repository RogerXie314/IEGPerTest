# 参与贡献指南 / Contributing Guide

欢迎参与 **IEGPerTest Simulator** 项目！本文档将帮助你了解如何提问、报告问题和贡献代码。

Welcome to **IEGPerTest Simulator**! This document will help you understand how to ask questions, report issues, and contribute code.

---

## 💬 如何提问 / How to Ask Questions

**当然可以提问！** 我们欢迎任何关于项目的问题。

**Yes, you can ask questions!** We welcome any questions about the project.

请通过以下方式提问：

1. **GitHub Issues**：[点击这里新建一个 Issue](../../issues/new/choose) 并选择 **"问题咨询 / Question"** 模板
2. 请尽量描述清楚你的问题背景，方便我们更好地帮助你

Please ask questions by:

1. **GitHub Issues**: [Click here to open a new Issue](../../issues/new/choose) and choose the **"问题咨询 / Question"** template
2. Please provide as much context as possible so we can help you effectively

---

## 🐛 如何报告 Bug / How to Report a Bug

如果你发现了 Bug，请通过 [GitHub Issues](../../issues/new/choose) 提交，并选择 **"Bug 报告 / Bug Report"** 模板。

If you find a bug, please submit it via [GitHub Issues](../../issues/new/choose) using the **"Bug 报告 / Bug Report"** template.

请在报告中包含：
- 复现步骤
- 期望行为与实际行为
- 操作系统和应用版本信息
- 如有截图，请一并附上

Please include in your report:
- Steps to reproduce the issue
- Expected vs. actual behavior
- OS and application version
- Screenshots if applicable

---

## ✨ 如何提交功能请求 / How to Request a Feature

如果你有好的想法，欢迎通过 [GitHub Issues](../../issues/new/choose) 提交，选择 **"功能请求 / Feature Request"** 模板。

If you have a great idea, submit it via [GitHub Issues](../../issues/new/choose) using the **"功能请求 / Feature Request"** template.

---

## 🛠️ 如何贡献代码 / How to Contribute Code

1. Fork 本仓库
2. 创建你的功能分支：`git checkout -b feature/your-feature-name`
3. 提交你的修改：`git commit -m 'Add some feature'`
4. 推送到分支：`git push origin feature/your-feature-name`
5. 开启一个 Pull Request
